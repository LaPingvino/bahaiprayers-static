Prayer ABU0030SHE in Esperanto

Ho Vi, pardonema Sinjoro! Vi estas la ŝirmo de ĉiuj ĉi Viaj servantoj. Vi konas la sekretojn kaj scias pri ĉiuj aferoj. Ni ĉiuj estas senhelpaj, kaj Vi estas la Potenca, la Ĉiopova. Ni ĉiuj estas pekantoj, kaj Vi estas la Pardonanto de pekoj, la Mizerikorda, la Kompatema. Ho Eternulo! Ne atentu niajn malperfektecojn. Traktu nin laŭ Via graco kaj malavaro. Multaj estas niaj malperfektecoj, sed senlima estas la oceano de Via pardonemo.
