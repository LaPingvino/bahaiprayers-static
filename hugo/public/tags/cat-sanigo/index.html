Prayer BH08013 in Esperanto

Ho Dio, mia Dio! Mi petegas al Vi je la oceano de Via kuraco, kaj je la brilo de la Tagstelo de Via favoro, kaj je Via Nomo, per kiu Vi subigis Viajn servantojn, kaj je la penetra forto de Via plej supera Vorto kaj la potenco de Via plej majesta Plumo, kaj je Via mizerikordo ekzistanta jam de antaŭ la kreo de ĉiuj en la ĉielo kaj sur la tero, purigu min per la akvoj de Via malavaro de ĉiu aflikto kaj malordo, kaj de ĉia feblo kaj malforto.

Thy name is my healing in Esperanto

Via nomo estas mia kuraco, ho mia Dio, kaj rememoro pri Vi mia medikamento. Proksimeco al Vi estas mia espero, kaj amo al Vi mia akompananto. Via mizerikordo al mi estas mia sanigo kaj mia helpo en tiu ĉi mondo kaj en la mondo venonta. Vi vere estas la Ĉio-dona, la Ĉio-scia, la Plejsaĝa. (Source category: Sanigo) (Bahaiprayers.net ID: 4311)
