Blessed is the Spot in Esperanto

Benita estas la punkto, kaj la domo, kaj la loko, kaj la urbo, kaj la koro, kai la monto, kaj la ŝirmejo, kaj la kavo, kaj la valo, kaj la lando, kaj la maro, kaj la insulo, kaj la kampo, kie oni Dion mencias kaj Lin laŭde gloras. (Source category: Laŭdo kaj Danko) (Bahaiprayers.net ID: 4337)

Prayer BH09401 in Esperanto

Ĉiuj laŭdoj, ho mia Dio, estu al Vi, Kiu estas la Fonto de ĉia gloro kaj majesto, de grandeco kaj honoro, de suvereneco kaj reĝeco, de alteco kaj favoro, de sankta timo kaj potenco. Kiun ajn Vi volas, Vi altiras al la Plej Sankta Oceano, kaj al kiu ajn Vi deziras, Vi donacas la honoron konfesi Vian Eternan Nomon. El ĉiuj, kiuj estas en la ĉielo kaj sur la tero, neniu povas rezisti la efikon de Via suverena volo.
